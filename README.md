# SalesAnalyzer
# 🛒 Sales Data Analytics Project (Core Python)

## 📊 Overview
This project is a **Core Python Data Analytics** case study on sales data.  
The goal was to analyze orders, revenue, and customer behavior using Python without external BI tools.  
It demonstrates skills in **data cleaning, manipulation, and analysis** with practical business queries.

---

## 🔍 Key Business Queries Solved
1. What is the total revenue from all sales?  
2. How many total orders were placed?  
3. How many total items (quantity) were sold?  
4. What is the total revenue generated by each category?  
5. What is the total revenue generated by each product?  
6. Which product has the highest quantity sold?  
7. Which customer spent the most money?  
8. What is the total revenue for each date?  
9. What is the total revenue between two given dates?  
10. Export all orders under the "Electronics" category into a new CSV file.

---

## ⚙️ Tech Stack & Skills
- **Python Core** (loops, conditionals, functions, file handling)
- **Libraries**: `datetime`, `csv`
- **Data Structures**: Lists, Dictionaries
- **File Operations**: Reading/Writing CSV
- **Problem Solving**: Business-oriented queries
